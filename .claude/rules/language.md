# Langue du projet

**IMPORTANT : Tout le projet est en français.**

## Règles

1. **Réponses** : Toujours répondre en français
2. **Explications** : Fournir les explications techniques en français
3. **Messages de commit** : En français
4. **Code** : Variables et fonctions en anglais (convention technique universelle)
5. **Commentaires dans le code** : En français
6. **Documentation** : En français
7. **Logs et messages utilisateur** : En français

## Seule exception

| Contexte                           | Langue  |
| ---------------------------------- | ------- |
| Code source (variables, fonctions) | Anglais |

Les noms de variables et fonctions restent en anglais car c'est la convention universelle du développement (compatibilité, lisibilité internationale du code).
