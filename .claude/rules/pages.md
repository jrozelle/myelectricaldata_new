# Documentation des Pages

**IMPORTANT : Pour toute modification de page, utiliser l'agent `frontend-specialist`.**

## Regle obligatoire

Chaque page dans `apps/web/src/pages/` doit avoir une documentation correspondante dans `docs/pages/`.

**Avant de creer ou modifier une page, consulter sa documentation dans `docs/pages/` si elle existe.**

**Apres avoir cree une nouvelle page, creer sa documentation dans `docs/pages/`.**

## Structure

| Page                                 | Documentation               |
| ------------------------------------ | --------------------------- |
| `apps/web/src/pages/Dashboard.tsx`   | `docs/pages/dashboard.md`   |
| `apps/web/src/pages/Consumption/`    | `docs/pages/consumption.md` |
| `apps/web/src/pages/Admin/Users.tsx` | `docs/pages/admin-users.md` |
